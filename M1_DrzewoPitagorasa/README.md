Projekt MI Drzewo pitagorejskie 
	
Wizualizacja drzewa pitagorejskie przedstawia animację tworzenia figury fraktalnej, przypominającej drzewo w formacie 2D.

Projekt został napisany na języku Python wersja 3.14.3, w edytorze Visual Studio Code.

Dla poprawnej wizualizacji aplikacji trzeba zainstalować biblioteki, instrukcja jest w tekstowym pliku requirements.txt 

Parametry do zmiany: głębokość gałęzi, prędkość animacji, stopień kątów.

Interfejs użytkownika za pomocą przycisków na klawiaturze: „P” pauza, „R” ustawienie domyślnego stopnia pewnego węgłu, „Space” start od nowa, „Esc” wyjście z aplikacji.
